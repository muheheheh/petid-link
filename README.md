## 项目概述

PetID Link 通过为每块宠物胸牌分配唯一二维码，建立宠物与主人的数字化连接。当宠物走失时，任何人扫码即可查看宠物信息、联系主人，并辅助追踪扫码位置。

---

## 核心业务流程

### 绑定流程（主人侧）

1. **购买/获取** — 主人获得实体胸牌，背面印有唯一二维码及遮盖式激活码。
2. **扫码激活** — 使用微信/App 扫码，系统识别该 ID 未绑定，跳转至激活页。
3. **身份验证** — 提示登录（微信/Google/Apple），登录后输入 6 位物理激活码。
4. **资料初始化** — 填写宠物信息（昵称、照片、品种、医疗史等），绑定成功。

### 扫码交互流程（访客侧）

1. **扫码触发** — 陌生人扫描宠物胸牌。
2. **静默上报** — 后端记录扫码 IP、时间、设备环境，并向主人推送实时提醒。
3. **展示详情** — 浏览器显示宠物资料页；若处于"丢失模式"，则高亮显示紧急联系方式。
4. **位置授权** — 引导访客点击"发送当前位置"，获取精确 GPS 坐标并更新至后台。

## 功能模块详细设计

### 用户与认证模块（Identity）

- **多方登录**
  - 国内：微信授权登录（WeChat OAuth）
  - 国外：Google Sign-In / Apple ID
- **账户管理** — 支持一个账号绑定多只宠物，支持账号注销与数据清理。

### 宠物与硬件管理（Device & Pet）

- **状态机切换**
  - `Normal`（正常）：仅展示基础信息，联系方式可选隐藏。
  - `Lost`（丢失）：页面变为红色警示主题，强制弹出位置授权请求，显示一键拨号按钮。
- **资料管理** — 支持上传宠物头像、录入过敏药物、疫苗接种状态等。

### 位置监控与告警（Tracking & Alert）

- **实时推送** — 二维码被扫描时，主人手机立即收到推送（App Push / 微信服务通知）。
- **位置记录**
  - 粗略定位：基于 IP 地址的城市级定位。
  - 精确定位：访客授权后的 GPS 经纬度。
- **足迹地图** — 管理端以时间轴 + 地图打点形式展示所有扫码记录。

### 安全与隐私控制（Security）

- **激活码机制** — 物理验证码防止二维码在流通过程中被恶意提前绑定。
- **信息脱敏** — 正常状态下隐藏主人手机号，通过中间号或平台留言交互。
