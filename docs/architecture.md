# PetID Link 基础设施架构

## 概览

当前阶段仅部署大陆节点，单台阿里云杭州 ECS 运行 K3s，所有服务容器化部署在集群内。

---

## 服务器

| 区域 | 位置 | 云商   | 数量 | 用途                     |
| ---- | ---- | ------ | ---- | ------------------------ |
| 国内 | 杭州 | 阿里云 | 1 台 | K3s 单节点，承载全部服务 |

---

## K3s 集群内服务

| 服务               | 说明                                          |
| ------------------ | --------------------------------------------- |
| 后端应用           | PetID Link 用户端 API 服务                    |
| PostgreSQL         | 关系型数据库，替代当前 SQLite                 |
| PostgreSQL MQ 插件 | 基于 PG 的消息队列（如 pgmq / LISTEN/NOTIFY） |
| Redis              | 缓存（Session、微信 access_token 等）         |

---
